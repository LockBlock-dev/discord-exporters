# Emotes


## Description

| File | Ban risk level |
| --- | --- |
| [Emotes](../modules/Emotes.js) | ![Low](https://img.shields.io/badge/-Low-brightgreen) |

Export all emotes from all guilds you are in.


## How it works :

The script requests the Discord API to get your entire list of guilds. Then it requests the list of emotes from each guild. Finally, it saves the results in a folder.
